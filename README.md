# Duality-Debugger
Archived version of a A short monochromatic platformer made for the Monochromatic Game Jam
Theme: “It’s not a bug, it’s a feature.”  
(Jam Page: https://itch.io/jam/monochromaticlittlejam)

## How to Play
- https://fisher08.itch.io/duality-debugger

## Gameplay
- Simple platforming mechanics
- Monochrome visual style
- From the second level the game starts to bug and glitch.
- Browser playable

## Made With
- Unity 6000.1.6f1
- Paint.net
- Audacity
